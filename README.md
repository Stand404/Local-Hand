## 功能
1. 局域网搜索、连接设备
2. 发送和接收文件、文字消息
3. 打开文件 (调出媒体播放器或选择其他方式弹窗)
4. 消息长按菜单 (复制文本、复制完整路径、打开文件夹)

## 安装

### Windows or Android
下载 [Releases](https://github.com/stand114514/Local-Hand/releases) 的编译版本

目前只有 Android 和 Windows 应用程序，其他平台请自行构建，开发人员没有设备测试
